# rollup-ts-dts-template-v2.33.1

#### master:

使用 rollup 的 typescript 模版, 使用插件同步生成 dts 文件

#### 使用方式:

1.修改项目名称和其 package.json 中"name"的名称

2.执行命令

```bash
npm install
```

3.打包

```bash
npm run build
```

4.测试

```bash
npm run test
```

#### 注意问题:

```
rollup-plugin-dts 插件版本过低会影响生成的 dts 文件, 经过插件测试在1.4.14版本可以
```
